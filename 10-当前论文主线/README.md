# 当前论文主线

这个目录放的是“现在写论文最该参考的版本”。

如果你只是想快速看懂项目，不用先翻历史聊天，也不用把所有旧日志读完。可以把这里当成一条已经整理好的阅读小路：先看直觉，再看口径，再看公式和术语。

推荐顺序：

1. [小白导读](./00-小白导读.md)：先知道问题为什么值得研究。
2. [当前论文口径](./01-当前论文口径.md)：写摘要、引言、方法时看这里。
3. [概念演化](./02-概念演化.md)：理解 TC-PPO、FTC、CTS、CSC 为什么一路改名。
4. [公式与算法速查](./03-公式与算法速查.md)：查公式、变量、算法模块。
5. [术语表](./04-术语表.md)：遇到缩写或概念混淆时查。
6. [下一步行动](./05-下一步行动.md)：看后续还要补哪些论文工作。

这条主线现在的意思是：

> 接触一致性帮我们早点发现训练在变坏；PPO 稳定机制尝试把训练拉回更可靠的接触状态。
