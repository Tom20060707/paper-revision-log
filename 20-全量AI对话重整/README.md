# 全量 AI 对话重整

这里是按日期整理过的 AI 对话摘要，不是第一阅读入口。

它的作用是保留研究过程：某个概念什么时候出现、为什么改名、哪些说法后来被推翻、哪些问题还没解决。真正写论文时，请先看 [当前论文主线](../10-当前论文主线/README.md)。

需要注意两点：

1. 每日日志保留了当时讨论的语境，所以里面可能出现旧目录名、旧方法名和旧判断。
2. 这些内容是研究过程证据，不等于最终论文口径。最终口径以 `10-当前论文主线/` 为准。

常用入口：

- [扫描报告](./00-扫描报告.md)
- [日期索引](./01-日期索引.md)
- [每日日志目录](./每日/)
