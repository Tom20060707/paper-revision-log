# 历史材料

这里放旧日志、旧索引和旧目录说明。

最重要的一点：`00-原始日志/` 现在保留了从 `2026-02-15` 到 `2026-05-01` 的按日记录，一共 53 篇。这里不能只留 4 月中下旬；前面 2 月、3 月和 4 月上半月的内容也要留在历史材料里，方便以后追溯。

它们不是删不掉的“垃圾文件”，而是用来追溯研究想法怎么来的。比如早期为什么叫 TC-PPO，为什么后来改成 CTS，再为什么升级到 CSC，都可以在这里找到痕迹。

但写论文时不要直接照抄这里的旧说法。尤其是这些内容需要谨慎：

- 把 TC-PPO 写成普遍提升所有任务的算法；
- 把 FTC/CTS 写成所有接触任务的通用指标；
- 把 `1 - CV²` 写成非周期操作任务也适用的总公式；
- 把 PBRS/Kitchen 写成已经解决的问题。

当前论文先看 [10-先读这些](../10-先读这些/README.md)。
