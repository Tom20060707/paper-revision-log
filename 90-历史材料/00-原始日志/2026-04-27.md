## 新洞察

1. **Kimi K2.6 是推理模型**：K2.6 输出分为 reasoning_content（思考链）和 content（最终答案），MCP 工具只读 content。max_tokens 不足时思考链耗尽 token，content 永远为空。正确用法：ask_kimi 至少传 max_tokens=8000，复杂任务传 12000。

2. **Abstract 末段定调效应**：Abstract 最后一句决定审稿人对全文的第一印象。以负面/限制性内容收尾，整篇会被归类为"分析失败案例的论文"而非"提出有效方法的论文"。

3. **Phi 函数两版本同时出现是定义一致性风险**：object-only Phi（PBRS 失败分析）和 approach Phi（Two-Phase 框架）若未明确区分，审稿人无法判断"PBRS 失败是原理问题还是 Phi 设计问题"。

4. **Zero-gradient barrier 的精确描述**：严格来说应为"zero-signal barrier"——非接触阶段物体状态不变导致 F(s,s')≈0，而非梯度本身为零。

## 修正的旧认知

1. **旧认知：Kimi API 不稳定**
   纠正：Kimi K2.6 API 完全正常，问题是 max_tokens 设置不当导致推理截断。moonshot-v1-8k 和 k2.6 在 max_tokens>=500 时均正常返回，max_tokens=5 时 content 为空。

2. **旧认知："invariance argument vacuous"是对 Ng 定理的合理批评**
   纠正：Ng 的 policy invariance 定理不要求 r_t 不等于 0，在 r_t=0 时数学上仍成立。正确批评是"practical alignment is no longer guaranteed"。
