# 图表与视觉叙事

## 图表目标

每张图都应回答一个审稿问题：

- Fig. 1：TC-PPO 是单一 policy 的训练侧调度，不是 HRL。
- Fig. 2：Layer-1/Layer-2 失效不同。
- Fig. 3：Mode B 是永久过正则化，不是暂时失败。
- Fig. 4：FTC 分布解释哪些任务落入 Mode A/B/C。
- Fig. 5：soft gating 比 hard gating 更稳，尤其体现方差下降。
- Appendix 图：补充设置和异常任务，不抢主线。

## 颜色语义

- 蓝色：vanilla/control baseline。
- 绿色：soft gating / Mode A 成功。
- 橙色：miscalibration / Mode B。
- 黄色背景：目标阈值区间或 Mode A window。
- 红色：崩溃区或警告区域。

## 删除原则

图表中不服务主线的算法列、过长 footnote、遮挡 legend 和没有解释的 N/A 都应该删掉。
