## 当前主线

论文当前最稳的定位是：

> 人形机器人 PPO 的困难不仅来自高维动作空间，也来自接触域退化。接触一致性可以作为崩溃前诊断信号，PPO 稳定机制则用于抑制退化并恢复接触域稳定性。

概念演化：

```text
TC-PPO / FTC
→ 接触丰富 PPO 的失稳诊断
→ CTS-PPO / Contact Timing Stability
→ Contact-Domain Consistency and Stability
→ CSC 总框架 + CTS 支撑接触特例
```

## 当前判断

- `TC-PPO` 是历史起点，不是当前唯一表述。
- `FTC/CTS` 不能被写成所有任务通用指标。
- `1 - CV²` 更适合作为准周期支撑接触的轻量实例。
- `CSC` 更适合作为总框架，用于描述任务相关接触面一致性。
- 一致性是诊断信号，稳定性是目标性质。
- Mode A/B/C 应解释为稳定域状态，不是任务好坏分类。
- Kitchen、insert、truck、push 等任务暴露的是非周期/阶段化接触建模边界。

## 数据边界

本仓库只整理已有对话和研究日志。所有实验数字仍必须回到真实服务器日志、图表脚本或论文源码验证。没有真实实验支撑的内容只能写成假设、边界分析、诊断解释或未来工作。
